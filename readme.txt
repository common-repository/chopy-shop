== Description ==

Chopy Shop is a Wordpress plugin to transform your blog in a E-commerce Shopping Cart, the install is very easy and in just a few seconds you will be ready to sell in your blog and accept payments with Paypal

This plugin inserts a shopping cart and buy buttons for your post, allow change the text so you can customize or translated into another language, also sends you an email every time someone places an order in your blog.

**In Spanish:**

Chopy Shop es un plugin para Wordpress que permite transformar tu blog en un carrito de compra de E-commerce, la instalación es bastante simple y en unos pocos segundos estaras listo para vender en tu blog y aceptar pagos con Paypal

Este plugin inserta un carro de compra y crea botones de compra para tus post, permite cambiar los textos de forma que puedas personalizarlo o traducirlo a otro idioma y te envia un correo electrónico cada ves que alguien realiza un pedido en tu blog.

== Installation ==

To install your Shopping Cart you need create a new Page and paste the next html code: [shopping-cart-chopyshop]<br />
Copy the URL of your Shopping Cart and paste in the option "Shopping Cart Page" in the general settings plugin.<br />
You also need create two new Pages more, One for a successful payment message and one for an incorrect payment.<br />
If you want change the text or translate the Shopping Cart, you can change the text from the "Shopping Cart Translate" options

**In Spanish:**

Para instalar tu carro de compra necesitas crear una nueva página y pegar el siguiente código html: [shopping-cart-chopyshop]<br />
Copia la URL de tu carro de compra y pégala en la opción "Shopping Cart Page" en las opciones generales del plugin.<br />
También necesitas crear dos nuevas páginas mas, una para el mensaje de pago correcto y otra para el pago incorrecto.<br />
Si deseas cambiar el texto o traducir el carro de compra, puedes cambiar el texto desde las opciones "Shopping Cart Translate"
